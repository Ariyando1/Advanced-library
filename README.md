# 📚 Advanced Book Discovery System

A fast and modern book search web application built with **React**, **Vite**, and **FlexSearch**.  
It provides an instant and smooth experience for exploring a large collection of books.
//static cover

## ✨ Key Features

- ⚡ **Blazing Fast Search** powered by FlexSearch for instant full-text indexing
- 🔍 **Multi-field search** across:
  - Titles
  - Authors
  - Summaries
  - Subjects
  - Bookshelves
- 🛠 **Smart filtering** by language
- 📊 **Sorting options** by download count or title
- 📖 **Book details modal** for deeper insights
- 📱 **Fully responsive UI** using Tailwind CSS

## 🛠 Tech Stack

- React
- Vite
- FlexSearch
- Tailwind CSS
